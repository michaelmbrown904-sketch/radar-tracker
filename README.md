# Radar Signal Processing and Tracking Pipeline

End-to-end pulse-Doppler radar simulation with detection and tracking,
implemented from scratch in Python.

## Structure
- `src/` — core modules (waveforms, radar, detection, tracking)
- `notebooks/` — exploratory Jupyter notebooks
- `tests/` — unit tests
- `figures/` — output plots

## Setup
```
conda create -n radar python=3.11
conda activate radar
pip install -r requirements.txt
```

## Status
In progress. See `notebooks/` for weekly exercises.
